Description:

This package contains a complete MenuBoard model with its textures and materials.
This package is optimised for mobile and VR applications.

Package includes:

- 1 MenuBoard 3D model (fbx format).
- 3 MenuBoard Prefabs
-----# menuboard = 36 tris
-----# Menuboard_double = 104 tris
-----# Menuboard_simple = 52 tris

- 3 diffuse board texture (jpg format) [512 x 256 pixels]
- 1 diffuse wood texture for body of the MenuBoard (jpg format) [512 x 256 pixels]

- 1 font
- 1 DEMO scene.
