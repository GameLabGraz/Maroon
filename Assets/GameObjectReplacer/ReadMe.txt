///////////////////////////////////////////////////////////////////
// GameObject Replacer is a quick tool to help replace multiple  //
// GameObjects in your scene with a prefab or a scene GameObject.//
///////////////////////////////////////////////////////////////////
//
////////////
// VIDEO //
//////////
// How To Use: https://www.youtube.com/watch?v=ZHJrom1XH9I
//
////////////
// USAGE //
//////////
//
// - Simple go to (GameObject or Window) > GameObject Replacer in the top Menu
// - Place your prefab or scene object in the Replacement GameObject field
// - Select 1 or more GameObjects in your scene
// - Choose if you want to keep rotation or scale
// - Press Replace GameObjects
//
